Blog Questions

Q-1. What is the Document Object Model (DOM)?

Answer:

The DOM is a programming interface for HTML and XML documents. It represents the page so that programs can change the document structure, style, and content dynamically. The DOM represents the document as nodes and objects, which allows languages like JavaScript to manipulate it.

Q-2. How do you select an element from the DOM?

Answer:

You can select an element from the DOM using methods such as getElementById, querySelector, and getElementsByClassName. These allow you to retrieve elements based on their ID, class, tag name, or CSS selectors.

Q-3. What is event delegation in the context of the DOM, and why is it useful?

Answer:

Event delegation is a technique in which a single event handler is attached to a parent element, allowing it to manage events triggered by its child elements. This is useful because it reduces the need for multiple event listeners, improving performance and making it easier to manage dynamic elements.

Q-4. How do you manipulate an element's attributes and styles using the DOM?

Answer:

You can manipulate an element's attributes using methods like setAttribute and getAttribute. To manipulate styles, you can directly modify the style property of an element, or by adding/removing classes using classList.add() and classList.remove().
